# directories and files

## list things out

### with `ls`

```bash
ls
ls -alh
```

### `tree`

```bash
tree
```

## create directories with `mkdir`

Create directory `toto`.

```bash
mkdir toto
```

Create hierarchy `toto/tutu`.

```bash
mkdir -p tutu/toto
```

## determine `file` type

```bash
file [file]
```

## `stat` on file

```bash
stat [file]
```
